Stabilizer Update Is Out! Get It Now! V1.6.2

Di Versi 1.6.2 ini, selain mendapatkan keseimbangan Exp Drop sesuai level player, juga tedapat beberapa fitur menarik lainnya.

Image by Novaskin Wallpaper and Edited by Yusril Takeuchi

[ V1.6.2 ]

- Add AddXP Command yaitu terdapat 1 Admin Command baru yaitu /levelup addxp untuk menambahkan XP seorang player lain ataupun sendiri.

- Add Top Player Level dimana kini telah terdapat Leaderboard level. Bisa dilihat dengan akses command /level top (Required RandomSK)

- Add Multiple Language Support tidak hanya bisa bahasa indonesia, kini LevelUp+ telah suppot bahasa internasional, inggris.

- Stabilizing Exp Drop Rate yaitu pendapatan Exp Player kini telah di seimbangkan, sesuai dengan player level itu sendiri.

Recommended Update Is Out! Get It Now! V1.6.1

Dalam V1.6.1 ini terdapat salah satu fitur update yang cukup spesial. Dimana terdapat Auto Max Level Setup. Kita tidak perlu mengcoding manual untuk menambahkan Max Level. Cukup mengganti value MaxLevel dengan angka sesuka hati, otomatis MaxLevel terupdate.

Image by Novaskin Wallpaper and Edited by Yusril Takeuchi

[ V1.6.1 ]

- Add AddXP Command yaitu terdapat 1 Admin Command baru yaitu /levelup addxp untuk menambahkan XP seorang player lain ataupun sendiri.

- Make Event On Tick Just For LevelUp untuk mencegah beratnya script saat diload. Yang membuat LevelUp+ lebih stabil

- Change Conditional Variable To Local dimana variable yang tadinya bersifaat sangat berat, diubah ke local hingga menjadi lebih ringan Dan nyaman untuk digunakan.

- Automatic Max Level Setup! adalah fitur yang paling spesial dalam update an kali ini. Kita tidak perlu mengcoding manual untuk menambahkan Max Level. Cukup mengganti value MaxLevel dengan angka sesuka hati, otomatis MaxLevel terupdate.

- Fixed Icon Menu Bug Sebelumnya ada beberapa icon di menu yang error, dan sudah terfix.

- Fixed Not Working Command telah memperbaiki beberapa command yang tidak bekerja dengan baik didalam game.

- Fixed Minus Exp When Death untuk lebih menstabilkan agar fitur ini jauh lebih baik bekerjanya.

- Fixed Error Sound Memperbaiki beberapa Sound yang error

- Removed version.yml Kurasa bagian ini tidak terlalu penting, hanya mempelambat proses load script

- Removed Junk Variable Menghapus beberapa variable sampah yang tidak berguna.

- Removed Auto Off Chat Prefix Menghapus auto Off Chat Prefix saat di reload.

Math Exp Drop Rate yang berguna untuk mengstabilkan pendapatan Exp sesuai level.

LevelUp+ Update To Stable Version V1.6!

Update V1.6 terdapat penambahan Max Level, Level Prefix Switch, Dan Fix beberapa bug. Dan juga penambahan Exp Drop Rate untuk Iron Golem, Wither, Player, dan Ender Dragon.

Image by Novaskin Wallpaper and Edited by Yusril Takeuchi

[ V1.6 ]

- Update Max Level 50 Di update V1.6 ini LevelUp+ telah melakukan Update Max Level dari 30 ke 50. Dan tentu dengan sistem Leveling yang menjadi lebih seret. Maksud seret disini tidak terlalu mudah untuk menaikan level hingga mencapai Level Max.

- Fixed Booster Status Error None Sebelumnya ketika LevelUp+ pertama kali dipasang, status Booster Status di /level mengalami error yaitu dan sekarang sudah di fix.

- Add LevelUp+ Level Prefix On/Off Sekarang LevelUp+ memiliki Level Prefix! Yap, ketika level chat di aktifkan, dan player melakukan chat, disana akan terpampang Level dari player tersebut. Dan jika dimatikan akan kembali seperti normal. Saat server di restart, Level Prefix akan ke disable secara otomatis, maka perlu di enable lagi untuk mengaktifkannya.

- Iron Golem, Wither, Player, Ender Dragon Now Can Be support Exp Booster Multiplier Di versi sebelumnya Iron Golem, Wither, Player, Ender Dragon tidak support dengan Exp Booster Multiplier yaitu booster untuk menggandakan EXP player. Tapi di versi ini, ke 3 mob ini dan player sudah bisa mendapatkan bonus XP saat efek Booster di aktifkan.

- Fixed Error Variable Can't Be Saved Di versi sebelumnya juga terdapat error variable {penyerang.%victim%} can't be saved. Dan sekarang sudah diperbaiki..

Bagi yang menginstall LevelUp+ V1.5c akan mendapatkan Update Notifikasi ketika join untuk disarankan melakukan Updating Skript... 1 Keluhan: Untuk mengload LevelUp+ V1.6 akan memakan waktu sekitar 6 menit.

Update To Version V1.5b

LevelUp+ telah update ke versi V1.5b sekarang! Dengan fitur-fitur yang baru antara lain :

[ V1.5b ]

- Add Hook System Hook sistem ini tambahan di versi ini. Jadi skript ini membutuhkan Add-on skript seperti SkQuery, Wildskript, dan ExtrasSk yang terinstall diserver kalian. Apabila ketiga addon ini tidak ada, maka skript akan didisable.

- Add Delay Menu Tambahan untuk mempercantik tampilan Menu. Diiringin dengan SFX yang tiap menu ter display, akan ada SFX tersendiri.

- Add More SFX Menu Ini saling berhubungan dengan Delay Menu. Ketika Delay Menu berjalan, maka SFX Menu ini juga akan berjalan.

- Add Admin Command Menu Update untuk Admin Command yang biasanya hanya menggunakan fitur message dari /levelup. Sekarang sudah ada tampilan menu yang tersusun lebih rapih.

- Add Join Message Ketika player memasuki server, akan ada notifikasi LevelUp+ yang tampil di bagian message.

- Add Particle Effect When Level Up Ketika player telah naik level, akan ada Particle firework yang diluncurkan ke player tersebut.

- Fixed Error Changelog Sebelumnya tampilan Changelog error, ketika dibuka, akan ada report error dari skriptnya. Namun sekarang sudah di fix.

- Fixed Exp Lost Error When Max Level Sebelumnya ketika sudah Max Level, dan mati. Exp Lost aktif, dan exp tetap berkurang ketika mati. Tapi sekarang sudah di fix. Ketika mencapai Max Level, dan mati. Exp tidak lagi berkurang.

- Fixed Menu Icon Display With Permission Tampilan Redeem Menu sebelumnya error, tertiban sama Changelog menu. Dan sekarang sudah diperbaiki..

Story Of LevelUp+

Image by Novaskin Wallpaper and Edited by Yusril Takeuchi

LevelUp+ awalnya bernama LevelUp yang diciptakan oleh Yusril Takeuchi yang secara tidak sengaja membuatnya. Pada saat itu dia sedang bosan, maka dari itu dia membuat skript LevelUp agar mendapatkan pekerjaan. Kemudian di Versi V1.2c seorang skriter bernama KinnaKikun berkeinginan ingin membantu melanjutkan project LevelUp, sehingga dia mengerjakan Version V1.2c dan mengubah nama LevelUp menjadi LevelUp+. Dan semenjak itulah, perkembangan skript ini terus di update oleh KinnakiKun dan Yusril Takeuchi sehingga bisa mencapai Version tertinggi sekarang yaitu V1.6
